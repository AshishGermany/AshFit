# AshFit

AshFit is a personal health, nutrition, fitness and wellness application built specifically around my own routines and goals.

Unlike commercial fitness apps, AshFit is designed to remove friction from everyday tracking by automating repetitive tasks and supporting personalized workflows.

## Current Features

- Nutrition tracking
- Food Library
- Favorites
- Autocomplete
- Daily calorie tracking
- Daily protein tracking
- Mobile-friendly interface

## Planned Features

- Gram-based nutrition
- Bone-in chicken calculations
- Workout logging
- Walking tracker
- Weight tracking
- Dashboard
- AI Insights

Version: v0.2.0 Alpha
